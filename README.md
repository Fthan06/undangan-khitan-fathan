## ⚙️ Kustomisasi

Semua config ada di bagian atas `main.js`:

```js

  eventDate : new Date('2026-06-11T08:00:00+07:00'),
  heroLines : ['MOCH.', 'FATHAN', 'RAFIF HAFIZHAN'],

  waNumber  : '628123456789',
};
```
